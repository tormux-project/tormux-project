## Nmap

### Installation
```
$ apt install nmap
```

### Run
```
$ nmap testphp.vulnweb.com
```

![nmap](https://i.ibb.co/5B8XGYJ/nmap.jpg)

* Port `80` open

* Try with `-A` _Enable OS detection_, _version detection_, _script scanning_, and _traceroute_
